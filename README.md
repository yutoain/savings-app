# 💰 貯金管理アプリ

カード払いと引き落とし日を管理できる、スマホ向け貯金管理PWAアプリです。

## 主な機能

### 📱 PWA対応
- iPhoneのホーム画面に追加してアプリのように使える
- オフラインでも動作
- アプリのようなUIで快適に利用可能

### 💳 カード管理
- 複数のクレジットカードを登録可能
- 各カードごとに締め日・引き落とし日を設定
- カード別の今月の使用額を表示
- 次回の引き落とし予定額を自動計算
- 今後3ヶ月分の引き落としスケジュールを表示

### 💸 支出管理
- 現金とカードの支払いを区別して記録
- カード払いの場合、引き落とし日を自動計算
- カテゴリ別に支出を分類
- カレンダー形式で支出履歴を表示
- 日別・月別の集計表示

### 🎯 貯金目標
- 目標金額と期限を設定
- カードの引き落としを考慮した予測
- 過去の平均支出から目標達成可能性を自動計算
- プログレスバーで進捗を視覚化

### 📊 ダッシュボード
- 今月の現金支出
- 今月のカード利用額（カード別）
- 次回引き落とし額の合計
- 引き落とし後の残高予測
- カード別使用状況を円グラフで表示

### 💬 モチベーションメッセージ
- カード引き落としも含めた収支で判定
- 目標達成度に応じたアドバイス
- カード使いすぎの場合は警告

### 🔔 通知機能
- 毎朝の予算通知（設定可能）
- 引き落とし日前の通知
- カードの引き落としを差し引いた金額で計算

### 🌙 その他の機能
- ダークモード対応
- データのエクスポート/インポート
- LocalStorageでデータ保存（サーバー不要）
- レスポンシブデザイン

## 使い方

### 1. アプリの起動

ブラウザで `index.html` を開くか、Webサーバーで公開してアクセスします。

**ローカルサーバーの起動例:**
```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx serve
```

その後、ブラウザで `http://localhost:8000` にアクセスします。

### 2. iPhoneのホーム画面に追加

1. Safariで開く
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

これでアプリアイコンがホーム画面に追加され、アプリとして使えます！

### 3. カードの登録

1. 画面下部の「⚙️ 設定」をタップ
2. 「カード管理」セクションで「追加」ボタンをタップ
3. カード情報を入力：
   - カード名（例：楽天カード）
   - 締め日（例：月末）
   - 引き落とし日（例：27日）
   - カラー（グラフ表示用）
4. 「保存」をタップ

### 4. 支出の記録

1. 画面下部の「💸 支出」をタップ
2. 支出情報を入力：
   - 金額
   - 日付
   - 支払い方法（現金 or カード）
   - カード払いの場合、引き落とし日が自動計算される
   - カテゴリ（任意）
3. 「記録する」をタップ

### 5. 目標設定

1. 画面下部の「🎯 目標」をタップ
2. 目標情報を入力：
   - 目標金額
   - 現在の貯金額
   - 目標期限
3. 「目標を設定」をタップ

アプリが自動的に：
- カードの引き落とし額を考慮
- 過去の支出傾向から予測
- 目標達成可能性を判定

### 6. ダッシュボードで確認

画面下部の「📊 ホーム」で以下を確認できます：
- 今月の支出状況
- カード別の使用額（円グラフ）
- 今後の引き落としスケジュール
- モチベーションメッセージ

## データ管理

### エクスポート
1. 「⚙️ 設定」→「データ管理」
2. 「データをエクスポート」をタップ
3. JSONファイルがダウンロードされます

### インポート
1. 「⚙️ 設定」→「データ管理」
2. 「データをインポート」をタップ
3. エクスポートしたJSONファイルを選択

## GitHub Pagesで公開する方法

1. GitHubリポジトリを作成
2. すべてのファイルをリポジトリにプッシュ
3. Settings → Pages → Source を `main` ブランチに設定
4. 公開されたURLにアクセス

**注意:** GitHub Pagesでは、`manifest.json`と`service-worker.js`のパスを調整する必要がある場合があります。リポジトリ名がURLに含まれる場合は、以下のように修正してください：

```json
// manifest.json
"start_url": "/リポジトリ名/",
```

```javascript
// service-worker.js
const urlsToCache = [
  '/リポジトリ名/',
  '/リポジトリ名/index.html',
  '/リポジトリ名/app.js',
  '/リポジトリ名/manifest.json'
];
```

## カスタマイズ

### カラーテーマの変更
[index.html](index.html) の `:root` セクションで色を変更できます：

```css
:root {
    --primary: #4F46E5;  /* メインカラー */
    --success: #10B981;  /* 成功カラー */
    --warning: #F59E0B;  /* 警告カラー */
    --danger: #EF4444;   /* 危険カラー */
}
```

### 通知時間の変更
設定ページで通知時間を選択できます。

## サポート

問題が発生した場合は、ブラウザのデベロッパーツールでコンソールを確認してください。

データはすべてブラウザのLocalStorageに保存されます。ブラウザのデータをクリアするとデータが失われるため、定期的にエクスポートすることをお勧めします。
